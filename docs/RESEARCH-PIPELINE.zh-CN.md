# 中国思想图谱科研数据流水线

版本：`method-v1`  
生效日期：2026-07-14

## 数据分层

- `data/research/` 是研究层，允许待提取、待核对、重复、存疑和被排除的记录。
- `data/*.csv` 是发布层，只能由 `npm run research:publish` 从研究层生成。

前端和 API 只读取发布层。人物候选、书目线索、研究状态和占位说明不得作为观点节点发布。

## 研究对象

观点节点必须是可归属于人物、可由证据支持、可独立赞成或反对的哲学命题。研究方向、学派名称、生平事实、著作目录、无论证功能的名言和网络语录不是观点。

人物可以是哲学家，也可以是对哲学问题产生实质影响的思想家、宗教家、科学家或政治思想家。仅凭知名度、身份或一句流行名言不足以纳入。

## 流水线

```text
人物登记 → 来源发现 → 著作与归属登记 → 原文摘录和定位
→ 候选命题切分 → 规范化命题 → 同人物内部去重/拆分
→ 中文审查 → 英文语义审查 → 关系证据审查
→ 发布门槛校验 → 生成前端 CSV/JSON/API
```

研究状态不得自动提升。语言模型、搜索摘要、普通百科和自动相似度只能帮助发现候选，不能直接产生已发布观点或历史影响关系。

## 研究层表

| 表 | 用途 |
|---|---|
| `people_status.csv` | 全部人物的纳入、优先级、研究阶段和下一门槛 |
| `person_sources.csv` | 人物与来源的候选映射，不等于内容已经核实 |
| `person_coverage.csv` | 按人物和思想维度追踪完整性与饱和度 |
| `works.csv` / `work_texts.csv` | 著作、年代和多语言题名 |
| `work_attributions.csv` | 作者、编者、注者、传统归名及其证据 |
| `source_excerpts.csv` | 原文摘录、精确定位、语言和核对状态 |
| `claims.csv` / `claim_texts.csv` | 规范命题及其中英文文本 |
| `claim_sources.csv` | 命题与原典/研究证据的精确连接 |
| `claim_decisions.csv` | 合并、拆分、重复和排除裁决 |
| `relations.csv` / `relation_texts.csv` | 观点关系和双语说明 |
| `relation_evidence.csv` | 关系的来源、定位与证据角色 |
| `exclusions.csv` | 被排除材料的原因留痕 |
| `review_events.csv` | 逐实体、逐维度的审核事件 |
| `release_manifests.csv` / `release_items.csv` | 可复现的发布版本清单 |

## 去重规则

- 同一人物、同一思想时期、真值条件相同：合并。
- 原文、现代汉语解释和英文翻译：属于一条观点的不同文本，不是三条观点。
- 可以被分别赞成或反对：拆分。
- 定义与由定义推出的规范结论：通常拆分。
- 同一人物前后期改变立场：分别保留，并记录时期。
- 跨人物的相似观点：不得合并，应建立相似、扩展、对照或反驳关系。

所有合并、拆分和排除决定都写入研究层，不能直接删除痕迹。

## 来源等级

1. 原典或可靠数字文本；
2. 校勘本、权威译本；
3. 同行评审的专门研究、大学出版社专著；
4. SEP、IEP、Oxford 等学术参考工具；
5. 人物机构主页、研究数据库；
6. 普通百科、搜索结果和聚合网页，仅用于发现。

传说人物、失传著作和传统归名必须明确记录历史性、作者归属和证据限制，不能把后世归名当作本人原话。

## 发布门槛

一条观点只有同时满足以下条件才可设为 `published`：

1. `content_type=position`；
2. 人物已纳入研究登记；
3. 有至少一个经过核对的来源定位；
4. 命题可独立判断，不是分类或人物简介；
5. 已完成同人物内部去重；
6. 中英文语义均已复核；
7. 审核人和日期完整；
8. 归属或年代争议已披露；
9. `research_status=canonical`；
10. 通过 `npm run research:validate`。

关系只有在两个端点均已发布、关系类型明确、关系说明双语完整且证据状态合格后才能发布。概念相似不得自动写成历史影响。

## 当前基线

截至 2026-07-14：155 人全部进入研究登记和公开观点层，形成 815 条人物—来源/资料库检索线索、756 条人物—思想维度覆盖记录、167 条逐条带来源定位的双语规范命题和 14 条观点关系。每位人物至少有1条观点；原有6位核心人物保留多观点结构。传说人格、后出语录、门人记录和严肃二手概括均在命题及归属字段中区分，不能伪装为本人同期逐字原话。

“来源线索已识别”不等于“完成独立同行复核”。当前167条均通过结构、双语、定位、去重与归属披露门槛；后续仍须按来源版本、翻译和人物体系覆盖扩充复核。覆盖报告必须分别显示人物登记数、已发布观点人物数、摘录数和待完成门槛。
